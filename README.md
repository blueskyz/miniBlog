miniBlog
========

使用 webpy 开发的迷你个人 blog。


功能：
-----

    1. 日志（wiki语法，支持代码高亮）
    1. 相册（上传，下载）
    1. 资源（上传，下载）
    1. 登录（根据用户权限大小，可以查看更多的资源，登录后可以使用管理功能）
    

部署：
-----

    # nginx 代理转发
    # uwsgi + webpy 网站
    # 数据库 mysql
    
    
下一步计划：
